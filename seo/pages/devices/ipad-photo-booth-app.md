# iPad Photo Booth App for Events

ChackTok supports iPad Photo Booth workflows for operators who need a portable event setup with templates, branded media, guest interaction, and sharing.

## When an iPad Booth Workflow Fits

iPad booth setups are useful for weddings, parties, retail activations, registration-adjacent experiences, and smaller venues where a full 360 platform is not required. Operators should confirm device model, app version, storage, camera permissions, stand stability, network, and charging before the event.

## Setup Checklist

1. Install ChackTok and log in.
2. Confirm iPadOS version, camera permission, media permission, storage, and power.
3. Create or clone an event template.
4. Upload branding assets and test the screen layout.
5. Test guest sharing on the event network.

## FAQ

### Can an iPad booth be used for professional events?

Yes, if the hardware, stand, lighting, event template, branding, and sharing workflow are tested before the event.

### What should operators test first?

Test camera quality, screen orientation, guest flow, gallery access, QR sharing, and battery or charging stability.
