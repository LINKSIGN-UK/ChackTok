# Photo Booth App for iOS and Android

ChackTok is a photo booth app for operators who need to turn supported mobile devices, booth hardware, and event templates into a repeatable commercial workflow.

## What the App Handles

The app supports event creation, device connection workflows, media capture, visual asset application, event sharing, and account-based operation. Operators can use it for 360 Photo Booth, iPad Photo Booth, GoPro, Magic Mirror, and other supported imaging scenarios.

## App Evaluation Checklist

| Question | Why it matters |
| --- | --- |
| Can the app reuse event templates? | Saves setup time across repeat events |
| Can guests receive media without friction? | Faster sharing improves guest and client satisfaction |
| Can branding assets be prepared before the event? | Reduces live-event mistakes |
| Can the setup work offline or with unstable WiFi? | Events often have weak networks |
| Can the app scale across devices? | Useful for operators running more than one booth |
