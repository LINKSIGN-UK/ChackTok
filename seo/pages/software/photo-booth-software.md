# Photo Booth Software for Event Operators

ChackTok is photo booth software for commercial event operators who need to create events, control supported booth hardware, capture branded photos or videos, and share media with guests from one workflow.

## What ChackTok Does

ChackTok helps operators run the practical parts of a photo booth event: event setup, supported device control, media capture, visual branding, gallery delivery, QR sharing, and post-event media management. It is built for event rental businesses, photo booth operators, imaging studios, and brand activation teams.

## Best Fit

| Operator need | How ChackTok helps |
| --- | --- |
| Repeatable event setup | Reusable templates and event workflows |
| Supported booth control | 360 Photo Booth, iPad Photo Booth, GoPro, and related workflows where compatible |
| Branded output | Overlays, frames, stickers, filters, music, and visual assets |
| Guest delivery | QR code, gallery, email, SMS where supported, and social sharing |
| Fleet growth | Mobile app workflows plus PC web console access |

## Basic Workflow

1. Download ChackTok from the App Store or Google Play.
2. Create or clone an event template.
3. Connect supported booth, camera, display, or sharing hardware.
4. Test media capture and guest sharing before guests arrive.
5. Run the event and manage output through the app or web console.

## FAQ

### Is ChackTok only for 360 Photo Booths?

No. ChackTok supports 360 Photo Booth workflows, iPad Photo Booth workflows, GoPro use cases, Magic Mirror scenarios, and other supported interactive imaging workflows.

### Does ChackTok replace event staff?

No. ChackTok supports the operator workflow. Staff should still test hardware, supervise guest safety, manage event settings, and handle client requirements.

### Where should operators start?

Start with the official tutorial page, then run a full test event on the exact device, network, and booth hardware planned for the commercial event.
