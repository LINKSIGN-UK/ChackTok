# 360 Photo Booth Software for Operators

ChackTok is 360 Photo Booth software for operators who need to prepare events, control supported 360 booth hardware, capture branded videos, and deliver guest media quickly.

## Why Operators Use It

360 events need more than a spinning platform. Operators must manage guest flow, booth control, camera setup, video effects, branding assets, sharing, and client delivery. ChackTok puts these tasks into one event workflow so teams can prepare before guests arrive and reduce live-event friction.

## Compatibility Notes

| Setup area | Notes |
| --- | --- |
| 360 booth control | Confirm model, controller, firmware, and app version before a paid event |
| GoPro workflows | Supported GoPro use cases should be tested with cable, wireless mode, storage, and battery |
| Mobile capture | iOS and Android workflows can vary by device capability |
| External screen | Confirm orientation, scaling, cable, or wireless display method |

## Setup Steps

1. Prepare the booth, phone or tablet, camera, power, and event network.
2. Create a ChackTok event or clone a tested template.
3. Connect the supported 360 booth and test start, stop, speed, direction, and duration controls where available.
4. Test capture, branding, and sharing with real event settings.
5. Keep backup power, network, cable, and device options ready.

## FAQ

### Can ChackTok control third-party 360 booths?

ChackTok supports hardware from more than 12 major Photo Booth brands through its wireless control architecture, subject to model, firmware, and setup conditions.

### Should operators test before every event?

Yes. Compatibility can vary by hardware, firmware, mobile device, operating system, cable, wireless environment, and app version.
