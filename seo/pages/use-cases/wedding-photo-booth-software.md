# Wedding Photo Booth Software

ChackTok helps wedding photo booth operators prepare event templates, capture branded photos or videos, and deliver guest media through a tested sharing workflow.

## Wedding Workflow Priorities

Wedding events need reliability, speed, and guest-friendly sharing. Operators should prepare couple names, event date, overlay style, gallery access, QR sharing, backup network, and a simple guest flow before the reception starts.

| Wedding need | ChackTok workflow |
| --- | --- |
| Custom branding | Names, date, overlay, frame, and visual assets |
| Fast guest delivery | QR code, gallery, email, SMS where supported |
| Staff handoff | Reusable event template and pre-event checklist |
| Event safety | Tested booth placement, device power, and connection stability |

## FAQ

### Can ChackTok be used for 360 wedding videos?

Yes, where the 360 booth, controller, device, firmware, and ChackTok app version are compatible. Operators should test the full capture and sharing workflow before the wedding.

### What should be prepared before wedding day?

Prepare event branding, music rights, sharing settings, booth safety, power, network, backup device, and a test gallery link.
