# Media Creation, Editing, and Background Music

ChackTok supports commercial event media workflows including capture, editing, saving, branding, and sharing.

## Supported Media Workflows

Depending on device capability and app version, ChackTok workflows may include:

- Photos.
- Videos.
- GIFs.
- Special-effect media.
- Overlays, frames, stickers, filters, and text.
- Background music.
- Gallery and sharing output.

## What to Prepare

- Final event branding assets.
- Overlay or frame files.
- Background music files with appropriate usage rights.
- Event name and sharing copy.
- Test device with enough storage.
- Stable network for upload and sharing.

## Create Event Media

1. Open the event in ChackTok.
2. Confirm the capture mode.
3. Connect booth, camera, or GoPro if required.
4. Capture a short test item.
5. Review framing, speed, lighting, and overlay position.
6. Adjust settings before guests arrive.

## Edit and Save Artworks

1. Open the captured media item.
2. Apply supported edits such as filters, overlays, stickers, or text.
3. Preview the output.
4. Save the final media.
5. Confirm that the saved item appears in the event gallery or media library.

## Upload Background Music

1. Prepare the audio file before the event.
2. Confirm usage rights for the event context.
3. Open the event or media settings area.
4. Upload or select the background music.
5. Preview playback with a test video.
6. Adjust volume if available.
7. Save and retest.

## Branding Best Practices

- Use clear final client assets.
- Avoid placing important logos near crop edges.
- Test output on the same device and sharing format guests will use.
- Keep a backup version without heavy effects in case performance is limited.
- Do not use unlicensed music or customer assets without permission.
