# Create and Manage ChackTok Events

This guide explains how to prepare common ChackTok event workflows.

## Event Types

Exact labels may vary by app version, but common workflows include:

- Party Event.
- Share Station Event.
- Auto Station Event.

## What to Prepare

- Event name.
- Date and venue.
- Branding assets.
- Overlays, frames, stickers, or text.
- Background music if used.
- Sharing method: QR code, gallery, email, SMS, or social channels.
- Hardware connection plan.
- Backup network plan.

## Create a Party Event

1. Open ChackTok and create a new event.
2. Choose the general party/event workflow where available.
3. Add event name and visual style.
4. Configure capture mode, branding assets, and sharing settings.
5. Save the event.
6. Run a test capture and confirm the output.

## Create a Share Station Event

1. Choose the Share Station workflow.
2. Configure the gallery or media access method.
3. Confirm how guests will access their media.
4. Test QR code, gallery, email, or SMS behavior before the event.
5. Place the sharing station where guests can use it without blocking capture.

## Create an Auto Station Event

1. Choose the Auto Station workflow.
2. Configure the capture and sharing sequence.
3. Set event assets and media output preferences.
4. Test the full workflow with the booth/camera connected.
5. Confirm the operator knows how to pause, restart, or exit the workflow.

## Template Best Practices

- Keep a tested base template for common event types.
- Duplicate and customize templates instead of building from scratch each time.
- Use final client logos and colors.
- Test media output on the actual event device.
- Keep a backup generic template for emergencies.

## What to Test

- Capture starts correctly.
- Media saves correctly.
- Overlay and music are applied correctly.
- QR code or gallery opens.
- Sharing text and links are correct.
- Event settings cannot be accidentally changed by guests.
