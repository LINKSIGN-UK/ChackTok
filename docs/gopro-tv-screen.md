# GoPro and TV Screen Setup

This guide covers practical preparation for GoPro and external screen workflows in ChackTok events.

## When to Use This Guide

- You are using a GoPro with a 360 Photo Booth.
- You need wired or wireless camera workflow guidance.
- You want to extend event visuals to a TV or external screen.

## What to Prepare

- Supported GoPro model.
- Updated GoPro firmware.
- Charged batteries or reliable power.
- Enough storage on the camera.
- Required cables or wireless setup.
- Phone/tablet running ChackTok.
- TV, projector, or external display if used.
- Test area with safe booth clearance.

## GoPro Setup Checklist

1. Charge the GoPro and clear storage.
2. Confirm the recording mode and resolution.
3. Update firmware if needed.
4. Connect the camera using the intended wired or wireless method.
5. Open ChackTok and confirm the event workflow.
6. Capture a short test media item.
7. Check whether the media imports, processes, and shares correctly.

## TV Screen Setup Checklist

1. Confirm whether the display uses wired or wireless output.
2. Test the phone/tablet connection to the TV or projector.
3. Check aspect ratio and orientation.
4. Run a sample event output on the screen.
5. Confirm guests and operators can see the intended content.
6. Keep backup cables/adapters ready.

## Common Problems

| Problem | What to check |
| --- | --- |
| GoPro not detected | Camera mode, firmware, battery, storage, cable/wireless setting |
| Recording fails | Storage, battery, resolution, overheating |
| TV does not show output | Cable/adapter, wireless display permissions, input source |
| Video transfer is slow | File size, network, camera connection quality |
| Output is cropped | Orientation, resolution, display scaling |

## Event-Day Advice

- Use lower recording settings if the device overheats or transfer is slow.
- Test the full workflow before guests arrive.
- Keep a second battery or power plan ready.
- Avoid changing camera settings during the live event unless necessary.
