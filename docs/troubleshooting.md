# ChackTok Troubleshooting

Use this guide when a live event workflow is not behaving as expected.

## First Response Checklist

1. Stay on the current event page and avoid deleting event data.
2. Check network, battery, storage, and app permissions.
3. Confirm the correct event/template is open.
4. Restart only the affected connection or device first.
5. Capture screenshots or screen recordings before changing too many settings.
6. Contact support if the issue persists.

## Bluetooth or Booth Connection

Check:

- Booth power.
- Bluetooth enabled.
- Device distance.
- Whether another phone/tablet is already connected.
- App permissions.
- Booth controller readiness.
- Model and firmware compatibility.

Try:

1. Turn Bluetooth off and on.
2. Power-cycle the booth.
3. Restart ChackTok.
4. Move closer to the booth.
5. Disconnect other paired devices.
6. Test the control page before restarting the event.

## Login or Account Issues

Check:

- Account spelling.
- Verification code.
- Network access.
- App version.
- Region-specific login restrictions.

If the account still fails, contact support with account details and screenshots.

## Upload or Sharing Issues

Check:

- Network stability.
- Whether the media finished processing.
- Whether the app stayed open long enough to upload.
- Gallery or sharing settings.
- QR code or link validity.
- SMS/email availability in the region.

Try a smaller test capture if files are very large.

## GoPro Issues

Check:

- Battery.
- Storage.
- Firmware.
- Recording mode.
- Cable or wireless connection.
- Camera temperature.

Use lower settings if transfer, processing, or overheating becomes a problem.

## Device Binding or VIP Issues

Check:

- Device number.
- Account used for binding.
- Network connection.
- Eligibility.
- Whether the app status refreshed after binding.

Contact support if a benefit is missing after binding.

## What to Send to Support

Send:

- ChackTok account.
- Device model and OS version.
- ChackTok app version.
- Booth/camera model.
- Event mode.
- Screenshots or videos.
- Time and region.
- What you already tried.

Support email: [support@chacktok.com](mailto:support@chacktok.com)
