# Getting Started with ChackTok

Use this guide when setting up ChackTok for the first time or preparing a new device for an event.

## When to Use This Guide

- You are installing ChackTok for the first time.
- You need to register or log in before an event.
- You are preparing a phone or tablet that will run an event.
- You need a quick pre-event setup checklist.

## What to Prepare

- A supported iOS or Android phone/tablet.
- Stable internet for login, download, upload, and sharing setup.
- ChackTok account information.
- Event name and basic event details.
- Branding assets if the event requires overlays, frames, text, music, or disclaimers.
- Hardware and camera devices if your workflow uses them.

## Download and Install

1. Open the official App Store or Google Play link from [OFFICIAL_LINKS.md](../OFFICIAL_LINKS.md).
2. Install ChackTok.
3. Open the app and allow required permissions when prompted.
4. Keep Bluetooth, camera, photos/media access, and network permissions enabled where your workflow requires them.

## Register or Log In

1. Open ChackTok.
2. Choose the sign-up or login entry.
3. Enter the required account information.
4. Complete verification if prompted.
5. Confirm that the account profile, VIP status, or device binding status loads correctly.

## Create a First Test Event

1. Create a new event or choose a template.
2. Add an event name.
3. Choose the event mode that matches the workflow.
4. Add overlays, frame assets, text, stickers, music, or sharing copy if needed.
5. Save the event.
6. Capture a short test item.
7. Confirm that the output can be saved, viewed, and shared.

## Pre-Event Checklist

- Login works.
- Event template is selected.
- Branding assets display correctly.
- Camera and/or booth connection is tested.
- QR code, gallery, email, SMS, or social sharing is tested.
- Device has enough storage and battery.
- Network plan is ready.
- Backup device or backup event workflow is ready.

## Common Issues

| Issue | What to check |
| --- | --- |
| Cannot log in | Account, network, app version, verification code |
| App cannot access camera/media | System permissions |
| Event media will not upload | Network, file size, app foreground state |
| Sharing link does not load | Upload status, gallery settings, network |
