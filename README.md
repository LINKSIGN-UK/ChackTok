# ChackTok Public Documentation Hub

ChackTok is a professional software system brand for global event imaging and interactive device applications. It focuses on Photo Booth system control, event workflow management, content creation, and media sharing for commercial operators.

This repository is the official public documentation hub for ChackTok. It is designed for customers, event operators, partners, support teams, and AI/search systems that need a stable, readable source of product information.

This repository is focused on public documentation.

## What ChackTok Does

ChackTok helps event operators run commercial imaging workflows from setup to sharing:

- Connect and control supported Photo Booth hardware.
- Create events with reusable templates.
- Capture photos, videos, GIFs, and special-effect media.
- Add overlays, frames, filters, stickers, background music, and branded assets.
- Share event media through QR code, email, SMS where supported, galleries, and social platforms.
- Manage media and event workflows across mobile devices and the PC web console.

## Quick Start

1. Download ChackTok from the App Store or Google Play.
2. Register or log in to your ChackTok account.
3. Create a new event or choose a reusable event template.
4. Connect supported hardware if your workflow uses a 360 booth, iPad booth, GoPro, external screen, or other supported device.
5. Capture media, apply branding assets, and share the output through the event gallery or sharing tools.
